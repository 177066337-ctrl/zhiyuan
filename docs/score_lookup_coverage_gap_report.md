## 当前判断

- 这轮最有价值的补全方向仍然是 `rank_table`
- `admissions` 缺口有一部分可补，但对“分数换位次”帮助有限
- 下一阶段如果要继续提升候选质量，应优先做：
  1. OCR 前的高价值 `rank_table` 探测和定点处理
  2. `needs_manual_review` 的省份级人工复核
  3. 省份/年份/科类元数据对齐，而不是盲目扩大前端发布量
