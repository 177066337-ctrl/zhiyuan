# 院校标签增强说明

## 本阶段目标

本阶段只做院校标签轻量增强，在不覆盖原始 `schools.json` 的前提下，为后续网页 MVP 提供更适合筛选的字段。

输出包括：

- `app/public/data/school_tags.json`
- `app/public/data/schools.enriched.json`
- `scripts/enrich_school_tags.py`
- `scripts/validate_school_tags.py`
- `docs/school_tag_validation_report.md`

本阶段不做：

- PDF 录取数据处理
- 推荐算法
- 网页开发
- OCR
- 大规模网页抓取

## 标签字段说明

增强后的院校记录包含以下新增或补充字段：

- `school_type`
- `is_985`
- `is_211`
- `is_double_first_class`
- `double_first_class_disciplines`
- `tag_sources`
- 重新汇总后的 `tags`

## 985 标签来源和处理规则

来源：

- 教育部“985工程”学校名单（2008-04-25）
- 页面链接：<https://www.moe.gov.cn/srcsite/A22/s7065/200612/t20061206_128833.html>

处理规则：

1. 以学校名称精确匹配为主。
2. 对已知名称差异使用 `aliases` 做补充映射。
3. 无法确认的不标记为 `true`。
4. 匹配来源写入 `tag_sources`。

## 211 标签来源和处理规则

来源：

- 教育部“211工程”学校名单（2008-04-25）
- 页面链接：<https://www.moe.gov.cn/srcsite/A22/s7065/200512/t20051223_82762.html>

处理规则：

1. 以学校名称精确匹配为主。
2. 对军队院校、新旧名称差异、已知别名使用 `aliases` 映射。
3. 无法确认的不标记为 `true`。
4. 匹配来源写入 `tag_sources`。

## 双一流标签来源和处理规则

来源：

- 教育部、财政部、国家发展改革委公布的第二轮“双一流”建设高校及建设学科名单（2022-02-11）
- 页面链接：<https://www.moe.gov.cn/srcsite/A22/s7065/202202/t20220211_598710.html>
- 本地解析文件：`tmp_double_first_class.pdf`

处理规则：

1. 先从官方 PDF 提取学校名称和建设学科。
2. 若学校能可靠匹配，则 `is_double_first_class = true`。
3. 若学科能可靠提取，则写入 `double_first_class_disciplines`。
4. 若学校可确认但学科文本不稳定，可保留空数组；本轮脚本优先保留能稳定解析出的学科列表。

## school_type 推断规则

`school_type` 不是事实性权威字段，而是前端筛选辅助字段。本轮采用名称关键词弱规则推断：

- 含 `师范` -> `师范`
- 含 `医科`、`中医药`、`药科`、`医学院`、`医药` -> `医药`
- 含 `财经`、`经济`、`金融`、`商学院`、`商业` -> `财经`
- 含 `理工`、`工业`、`科技`、`工程`、`电子科技`、`邮电`、`电力`、`海事`、`铁道`、`铁路` -> `理工`
- 含 `农业`、`农林`、`林业` -> `农林`
- 含 `政法`、`公安`、`警察`、`司法` -> `政法`
- 含 `外国语`、`语言` -> `语言`
- 含 `美术`、`音乐`、`艺术`、`戏剧`、`电影`、`传媒`、`舞蹈` -> `艺术`
- 含 `体育` -> `体育`
- 含 `民族` -> `民族`
- 含 `军` -> `军事`
- 其余本科大学名称 -> 弱规则回退为 `综合`
- 其余未命中 -> `其他`

局限：

- 该规则不等同于教育部正式院校类型分类。
- 一些院校具有复合属性，名称无法完整反映办学特色。

## ownership 处理规则

1. 优先沿用 `schools.json` 中已有的 `ownership`。
2. 若原字段为空，则回退为 `其他`。
3. 本阶段不做大规模新增推断。

## aliases 别名映射规则

本阶段仅维护少量高价值别名映射，主要解决：

- 军队院校新旧名称差异
- 医学院校旧称差异
- 常见简称和正式名称差异

当前典型示例：

- `国防科学技术大学` <- `中国人民解放军国防科技大学`、`国防科技大学`
- `第二军医大学` <- `中国人民解放军第二军医大学`、`海军军医大学`
- `第四军医大学` <- `中国人民解放军第四军医大学`、`空军军医大学`
- `北京协和医学院` <- `中国协和医科大学`

## 已知局限

1. `schools.json` 来源于 2016 年普通高校名单，不包含部分军队院校或后续更名信息，因此官方 985/211/双一流名单与当前院校底表可能存在差集。
2. `school_type` 是弱规则推断，准确率有限。
3. `official_site` 仍为空，尚未补充官网源。
4. `985/211/双一流` 标签尽量保守，但仍建议后续做人工复核。

## 后续人工校对建议

1. 优先核对未出现在 `schools.json` 中的官方 985/211/双一流学校。
2. 重点检查 `school_type = 其他` 的本科院校。
3. 补充高价值院校标签：
   - `985`
   - `211`
   - `双一流`
   - 官网
   - 标准院校类型
4. 若下一步做专业查询体验优化，建议优先补 `subject_requirement`（选科要求）。
